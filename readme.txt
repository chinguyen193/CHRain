Generating hourly rainfall spline using ANUSPLIN program
1. Disagg_d2h.py : python script to disaggregate daily data to hourly data using the pattern from the nearby station
2. Prepare_hourly_cmt.py : prepare cmt file to run ANUSLIN program. 
3. Prepare_hourly_input.py : prepare input .dat file to run ANUSPIN program.
